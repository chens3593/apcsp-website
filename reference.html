Name of Command Brief Description Usage example
cd This command changes the current working directory The command (cd scripts) will change the directory to scripts
ls This lists information about the things inside the current directory (ls -al) lists all the things
nano This opens a script for the file using nano (nano myfirstbashscript) will open myfirstbashscript
echo This makes the computer say whatever follows (echo Hell World) will say Hello World
man This displays a manual of how to use the following command (man ping) will show how a ping can be modified
cat This displays the content of the following file (cat myfirstbashscript) will show the code in myfirstbashscript
scp This copies the file first mentioned to the location mentioned next. (scp myfirstbashscript scripts) will move myfirstbashscript to scripts
rm This removes a certain object (rm myfirstbashscript) will remove myfirstbashscript
mkdir This makes a directory (mkdir scripts) will make a directory called scripts
pwd This prints the working directory (pwd) will print the working directory
